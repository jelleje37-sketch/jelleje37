## 🎓 Capstone Project Framework

All projects in this portfolio follow a **professional 13-section framework** that ensures consistency, clarity, and recruiter-friendliness.

### Framework Components:

1. **Project Title & Summary** – Context and business value
2. **Business Problem** – Questions, stakeholders, success metrics
3. **Data Preparation** – Source, size, scope, dictionary
4. **Data Processing** – Cleaning, validation, feature engineering
5. **Analysis** – Findings, insights, statistical evidence
6. **Visualization** – Charts, dashboards, executive summaries
7. **Recommendations** – Actionable next steps by stakeholder
8. **Technical Setup** – Repository structure, reproducibility
9. **Skills Demonstrated** – Technical and analytical capabilities
10. **Future Improvements** – Scalability and extensions

📖 **Full Guide:** See [CAPSTONE_TEMPLATE_GUIDE.md](./CAPSTONE_TEMPLATE_GUIDE.md)

## 📊 Portfolio Projects

### 🍎 **Metabolic Health & Ultra-Processed Food Audit**

**Business Question:** How does ultra-processed food consumption relate to metabolic health and adverse health outcomes?

**Key Highlights:**
- Analyzed **261,000+ FDA adverse event records**
- Built multi-CTE BigQuery SQL pipeline
- Identified demographic risk patterns
- Applied NOVA food classification framework

**Tools:** BigQuery, Python, Pandas, Plotly  
**Stakeholders:** Public health officials, nutrition teams, healthcare administrators  
**Impact:** Risk-based intervention recommendations by demographic

📁 **Location:** `bigquery-sql/health-nutrition/`  
📓 **Kaggle Notebook:** [2026-audit-ultra-processed-foods-metabolism](https://www.kaggle.com/code/lunorabloom/2026-audit-ultra-processed-foods-metabolism)

---

### ⌚ **FitLife360 Wellness Audit**

**Business Question:** What trends emerge from wearable biometric data that could improve wellness interventions?

**Key Highlights:**
- Processed population-level wearable health metrics
- Created activity, sleep, and metabolic trend visualizations
- Demonstrated health-tech analytics capability
- Built regional wellness benchmarking model

**Tools:** R, tidyverse, ggplot2, BigQuery  
**Stakeholders:** Wellness program managers, clinical teams  
**Impact:** Data-driven wellness intervention strategies

📁 **Location:** `wearables-fitlife360/`  
📓 **Kaggle Notebook:** [Wellness Audit - FitLife360 EDA Insights](https://www.kaggle.com/code/lunorabloom/wellness-audit-fitlife360-eda-insights)

### 🔬 **Cirrhosis Survival Analysis Dashboard**

**Tools:** Tableau, Excel, Statistical analysis, Python  
**Stakeholders:** Hepatology team, patient care coordinators  
**Impact:** Evidence-based patient risk assessment and prioritization

📁 **Location:** Portfolio section  
📓 **Kaggle Notebook:** [Cirrhosis Survival Analysis and Predictive Model](https://www.kaggle.com/code/lunorabloom/cirrhosis-survival-analysis-and-predictive-model)

### To Start a Healthcare Analytics Capstone Project:

1. **Copy the template**
   ```bash
   cp -r capstone-project/ [new-project-name]/
   ```

2. **Follow the README in your new folder**
   - Fill in all 13 sections
   - Adapt SQL templates
   - Run Jupyter notebooks

3. **Reference existing projects** for examples
   - See how findings are documented
   - Review visualization approaches
   - Study recommendation frameworks
