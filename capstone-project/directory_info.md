# 📂 Capstone Project Directory Guide

This is the root folder for your Data Analytics Capstone Project.

## 📂 Folder Structure

```
capstone-project/
├── data/
│   ├── raw/              # Original dataset files
│   └── processed/        # Cleaned and processed data
│
├── notebooks/            # Jupyter notebooks for analysis
│   ├── 01_EDA_and_Cleaning.ipynb
│   ├── 02_Analysis.ipynb
│   └── 03_Visualizations.ipynb
│
├── sql/                  # SQL queries for data processing
│   ├── data_cleaning.sql
│   ├── feature_engineering.sql
│   └── analysis_queries.sql
│
├── tableau/              # Tableau workbooks and exports
│   ├── dashboard.twbx
│   └── dashboard_screenshots/
│
├── images/               # Charts, dashboards, visualizations
│
├── README.md             # Main project documentation
└── LICENSE               # License file
```

## 🚀 Quick Start

1. **Add your raw data** to `data/raw/`
2. **Run cleaning script** from `sql/data_cleaning.sql`
3. **Explore the data** using `notebooks/01_EDA_and_Cleaning.ipynb`
4. **Conduct analysis** with `notebooks/02_Analysis.ipynb`
5. **Create visualizations** using `notebooks/03_Visualizations.ipynb` or Tableau
